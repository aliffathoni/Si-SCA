# ECG Example

This code provide example to get ecg graph data from AD8232 Heart Rate Sensor and display it using Serial Monitor and/or Serial Plotter on Arduino IDE.

|FUNCTION| ESP32  | AD8232 |
| ------------- | ------------- | ------------- |
| Leads Out +  | 16 | LO+ |
| Leads Out -  | 17 | LO- |
| Analog Output  | 34 | Output |
| V Input | 3.3 | 3.3 |
| Ground | NC | GND |
